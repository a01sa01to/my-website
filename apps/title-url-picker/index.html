    その名の通りのChrome拡張機能。開いているタブのタイトル・URLを取得でき、またMarkdown形式でリンクを作成できる。

    A Chrome extension as the name suggests. It can retrieve the title and URL of an open tab, and create a link in Markdown format.

    使い方

    ページのタイトルを取得

    1. 拡張機能を開きます。
    2. 「Get Title」ボタンをクリックします。
    3. テキストボックスから、タイトルをコピーできます。

    ページのURLを取得

    1. 拡張機能を開きます。
    2. 「Get URL」ボタンをクリックします。
    3. テキストボックスから、URLをコピーできます。

    Markdown形式でページへのリンクを作成

    1. 拡張機能を開きます。
    2. 「Get both with Markdown」ボタンをクリックします。
    3. テキストボックスから、リンクをコピーできます。

    導入方法

    • Chrome Webstoreからインストールしてください。（まだできません）

    How to Use

    Get the Page's Title

    1. Open the Extension.
    2. Click "Get Title"
    3. Copy the Title from the Textbox with the button.

    Get the Page's URL

    1. Open the Extension.
    2. Click "Get URL"
    3. Copy the URL from the Textbox with the button.

    Create a link to the Page with Markdown Format

    1. Open the Extension.
    2. Click "Get both with Markdown"
    3. Copy the Link from the Textbox with the button.
