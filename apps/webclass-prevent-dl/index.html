    Show my report without downloading (WebClass)

    1. Home
    2. アプリ一覧
    3. このアプリ
    1. Home
    2. My Apps
    3. This App

    WebClassで、提出したレポートなどをダウンロードせずに閲覧するChrome拡張機能。

    A Chrome extension that allows you to view submitted reports and other information in WebClass without downloading them.

    このアプリについての説明

    WebClassを使う際、 `（科目）→「マイレポート」→（ファイル名）` とすることで提出したレポートを見ることができます。
    しかし、閲覧するためにはダウンロードする必要があります。
    この拡張機能を用いると、ダウンロードすることなくファイルを閲覧することができます。

    なお、このアプリはスコープの関係上、埼玉大学のWebClassにおいてのみ利用できます。

    使い方

    インストールするだけです。

    導入方法

    • Chrome Webstoreからインストールしてください。

    Description about this application

    .

    When you use WebClass, you can view your submitted reports by going to `(subject) -> 'My Reports' -> (file name).
    However, you will need to download the file to view it.
    This extension allows you to view the file without downloading it.

    Because of its scope, this application can only be used in WebClass of Saitama University.

    How to Use

    Just install it

    Installation

    • Download from Chrome Webstore.
