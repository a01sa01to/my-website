当サイトのオープンデータカタログでは、APIをご利用いただけます。このページでは、本APIの利用方法をご案内します。

You can use the API in the Open Data Catalog on this website. This page will guide you through the use of this API.

If you want to see how to use the previous API, please check here. Please be aware that the API will not be available after March 1, 2022.

基本仕様

Basic Specifications

  • a01sa01to.comドメイン配下で提供されます。APIについても同様です。
  • HTTPS で提供されます。
  • 日付等はすべて日本時間です。
  • Header には Access-Control-Allow-Origin:* が設定されています。
  • All Open Data are provided under the a01sa01to.com domain and HTTPS. The same is true for APIs.
  • All dates are in Japan time.
  • "Access-Control-Allow-Origin:*" is set in the Header

利用方法

How to Use

JSON形式で取得する

当APIはGraphQLを用いており、実際に Playground でお試しいただけます。Playground で cURL のコードをコピーできます。エンドポイント（https://a01sa01to.com/opendata/api/）にGET/POSTリクエストしても、データを取得できます。
なお、使い方については随時変更予定で、現在不足している使い方等についても追記する予定です。

This API uses GraphQL, and you can actually try it out in the Playground. You can copy the cURL code in Playground. You can also retrieve data by making a GET/POST request to the endpoint at https://a01sa01to.com/opendata/api/.
We are planning to change the usage as needed, and will also add information about the current lack of usage.

生のデータを取得する

Get the raw data

  1. 利用したいオープンデータのページにアクセスします。


  2. 「データへのURLを取得する」ボタンをクリックします。


  3. モーダルが表示されます。「URLをコピー」ボタンをクリックすると、URLがクリップボードにコピーされます。
  1. Access the page of the open data you want to use.


  2. Click the "Get URL to the data" button


  3. A modal will be displayed. Click the "Copy the URL" button to copy the URL to the clipboard.
