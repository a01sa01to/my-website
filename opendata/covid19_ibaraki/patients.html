    茨城県内の、新型コロナウイルス感染症の陽性患者の年代、性別、居住地などの属性情報を一覧化したオープンデータです。

    This is open data that lists the age, gender, residence, and other attributes of positive patients for COVID-19 infection in Ibaraki Prefecture.


    当データを利用した作品を公開する際には、必ず当ページへのリンクを記載してください。

    When you publish a work using this data, please be sure to include a link to this page.

    • 当データの情報の正確性については万全を期していますが、利用者が当サイトの情報を用いて行う一切の行為について責任を負うものではありません。また、陽性者の特定等を目的とするものではありません。
    • 当データは、あくまで県の公式情報をもとに作成したものです。県によって公式に公開されているものではありません。
    • 情報の正確性を万全に期すため、当データでは、原則として県が公表しているデータ以上の情報を掲載していません。しかし2020年12月25日以降は、県の情報がかなり曖昧なものになったため、茨城新聞社によるデータも用いています（そのデータには文末に「？」をつけています）。
    • 施設名の繰り返しを避けるために「○○例目の施設」と表記しています。決してその全員が「○○例目」の方から感染が広がったとは限りません。ご理解ください。
    • 公表されていない部分は空欄になっています。記入漏れというわけではありませんので、ご理解の程よろしくお願いします。
    • CSVファイルをダウンロードした際にExcelで表示すると、文字化けが起きたり、列が異なったりすることがあります。その際には、別のソフトウェアをご利用ください。
    • This data has been released as open data in accordance with the format used in the "Open Data Item Definitions for COVID-19 Infection Control" and is open to the public as open data.
    • While every effort has been made to ensure the accuracy of the information in this data, we are not responsible for any actions taken by users using the information on this site. Also, the purpose of this data is not to identify patients.
    • This data is based on the official information of the prefecture, not officially published.
    • In order to ensure the accuracy of the information, this data does not, in principle, contain more information than the data published by the prefecture. However, since 2020-12-25, the information from the prefecture has become quite vague, so the data from Ibaraki Shimbun is also used (the data is indicated with a "?" at the end of the sentence).
    • To avoid repeating the name of the facility, we use the term "XXth case facility". By no means does it mean that all of them have spread the infection from the "XXth case". Please understand this.
    • The unpublished part is left blank. Please understand that this does not mean that the information has been omitted.
    • If you download a CSV file and display it in Excel, you may encounter garbled characters or different columns. In this case, please use another software.
