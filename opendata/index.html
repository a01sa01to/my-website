    About this site

    このサイトは、Asaがオープンデータを提供するために立ち上がったサイトです。
    公式には提供されていないデータを、Asaがまとめて提供します。
    ファイル形式はCSVやJSONを用意しています。

    This site has been set up by Asa to provide open data.
    I'll compile provide data that is not officially provided.
    CSV and JSON file formats are available.

    What is Opendata?

    オープンデータとは、営利目的、非営利目的を問わず二次利用可能なルールが適用され、機械判読に適しており、無償で利用できるデータのことです。
    当サイトで公開しているデータの情報の正確性については万全を期していますが、利用者が当サイトの情報を用いて行う一切の行為について責任を負うものではありません。

    Opendata is data that is subject to rules that allow secondary use for both commercial and non-commercial purposes, is suitable for machine reading, and can be used free of charge.
    While every effort has been made to ensure the accuracy of the information in the data published on this site, we cannot be held responsible for any actions taken by users using the information on the site.
